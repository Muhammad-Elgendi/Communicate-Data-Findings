Bay Wheels In August 2020 Exploration¶

by Muhammad Elgendi¶

Preliminary Wrangling¶

Bay Wheels is a regional public bicycle sharing system in California's San Francisco Bay Area. Bay Wheels is the first regional and large-scale bicycle sharing system deployed in California and on the West Coast of the United States. This dataset was collected from trips in august 2020 and it has 152446 records.

Each trip has the following features:

  • Ride ID
  • Bike type
  • Start Time and Date
  • End Time and Date
  • Start Station Name
  • Start Station ID
  • End Station Name
  • End Station ID
  • Start Station Latitude
  • Start Station Longitude
  • End Station Latitude
  • End Station Longitude
  • User Type (Subscriber or Customer Subscriber = Member or Customer = Casual)

aug20['duration_sec'] = (aug20['ended_at'] - aug20['started_at']).dt.total_seconds()

What is the structure of your dataset?¶

The original dataset has 13 variables and around 150000 records. These variables can be catagorized into :

  1. Bike info : ride_id and rideable_type
  2. trip time : started_at and ended_at
  3. station info : start_station_name, start_station_id, end_station_id, end_station_name, start_lat, start_lng, end_lat and ent_lng.
  4. user info : member_casual variable

I drived other feature from trip time variables for analysis:

  1. duration_sec (trip duration in seconds)
  2. duration_min (trip duration in minutes)
  3. start_date
  4. start_hour
  5. start_day

What is/are the main feature(s) of interest in your dataset?¶

started_at, ended_at and member_casual variables

What features in the dataset do you think will help support your investigation into your feature(s) of interest?¶

started_at, ended_at and all drived variables will help me understand the usage patern in terms of time and questions like :

  • When are most trips taken in terms of time of day, day of the week?
  • How long trip take in average ?
  • Does the above depend on if a user is a subscriber or customer?
  • Does user type has any relation with rideable type ?
